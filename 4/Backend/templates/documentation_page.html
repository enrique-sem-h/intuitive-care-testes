Essa API foi criada para o teste de API para a vaga de estagio na Intuitive Care

Atualmente conta com 2 endpoints

'/items' -> Retorna todos os itens da tabela de dados cadastrais das operadoras ativas na ANS

'/items/search:<parametro>' -> Retorna os itens mais relevantes da tabela para a pesquisa textual <parametro>

OBS: A busca retorna os itens em todas as colunas
